# API 模块

此目录包含应用程序的所有 API 路由定义。每个文件处理与系统不同方面的特定功能。

## 文件说明

- `account_management_routes.py` - 账户管理操作路由
- `account_routes.py` - 核心账户相关路由
- `config_routes.py` - 配置相关 API 端点
- `crypto_routes.py` - 加密货币特定 API 路由
- `market_data_routes.py` - 市场数据获取和处理路由
- `order_routes.py` - 订单下单和管理路由
- `ranking_routes.py` - 排名和排行榜相关路由
- `user_routes.py` - 用户管理和认证路由
- `ws.py` - 用于实时通信的 WebSocket 实现
