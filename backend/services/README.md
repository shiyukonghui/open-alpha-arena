# 服务模块

该目录包含业务逻辑和服务实现。

## 文件说明

- `ai_decision_service.py` - AI决策服务
- `asset_calculator.py` - 资产计算服务
- `asset_curve_calculator.py` - 资产曲线计算服务
- `auto_trader.py` - 自动交易服务
- `hyperliquid_market_data.py` - Hyperliquid市场数据服务
- `market_data.py` - 市场数据服务
- `news_feed.py` - 新闻订阅服务
- `order_executor_leverage.py` - 杠杆订单执行服务
- `order_matching.py` - 订单匹配服务
- `order_monitor.py` - 订单监控服务
- `order_scheduler.py` - 订单调度服务
- `price_cache.py` - 价格缓存服务
- `scheduler.py` - 任务调度服务
- `startup.py` - 启动服务
- `trading_commands.py` - 交易命令服务
